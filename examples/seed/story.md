# Seed Story

## Title

As a registered user, I want to reset my password so that I can regain access to my account.

## Acceptance Criteria

- The login page contains a "Forgot Password" link.
- Submitting a registered email triggers a password reset email.
- The reset email contains a secure reset link.
- The reset link allows submitting a new password.
- The user can sign in with the new password after reset.
